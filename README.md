# 🚀 Career Co-Pilot: Your AI-Driven Career Mentor

**Transforming Resumes into Readiness.**

Career Co-Pilot is a high-fidelity, hackathon-ready platform designed to bridge the massive gap between student profiles and industry-ready professionals through AI-powered skill extraction and personalized upskilling roadmaps.

---

## 🏆 The Problem
Students and early professionals often struggle with:
- **Directionless Learning**: Wasting months on irrelevant courses.
- **The "Missing" Link**: Not knowing exactly which skills to focus on for their dream role.
- **Static Resumes**: Resumes that don't effectively communicate technical readiness to recruiter systems.

## ✨ Our Solution: Career Co-Pilot
Our platform acts as a smart career coach, guiding you from your current skill level to a job-ready profile in 4 easy steps:

1.  **AI Resume Parsing**: Instantly extract technical & soft skills directly from any PDF resume.
2.  **Target Role Matching**: Select your goal (Frontend, Backend, AI/ML, etc.) and see how your profile stacks up.
3.  **Real-Time Gap Analysis**: Discover exactly what's missing and get a "Readiness Score."
4.  **Personalized Roadmap**: Get a dynamic, 4-5 week week-by-week learning path with curated resources.

---

## 🛠️ Tech Stack
- **Frontend**: Vite + React (Fast & Modern)
- **Styling**: Tailwind CSS (Premium UI aesthetics)
- **Animations**: Framer Motion (Smooth layout transitions)
- **Parser Core**: PDF.js (Client-side localized text extraction)
- **Iconography**: Lucide-React

## 🧠 Key Features & MVP Components
- **Live PDF Extractor**: Uses a client-side parser to read resume text without needing a backend.
- **Dynamic Analysis Engine**: Calculates readiness scores and detects gaps based on the *actual* content of the uploaded PDF.
- **Interactive Roadmap**: A milestone-based learning path that adapts to your specific skill gaps.
- **Hackathon-Ready Verification**: A verification step that lets users confirm AI findings before generating results.

---

## 🚀 Future Vision

## 🏁 Getting Started
### 📦 Prerequisites
- **Node.js**: v18+ 
- **npm** or **yarn**

### 💻 Installation
1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/Pranayepuru/NextStep-AI.git
    ```
2.  **Navigate to Project**:
    ```bash
    cd "carrer copilot"
    ```
3.  **Install Dependencies**:
    ```bash
    npm install
    ```
4.  **Run Development Server**:
    ```bash
    npm run dev
    ```
5.  **Open in Browser**: [http://localhost:5173](http://localhost:5173)
